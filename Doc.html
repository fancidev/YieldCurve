Description of yield curve models and numeric methods.

A yield curve model takes as input a set of observed swap rates of different maturities on a given day and produces as output a swap curve that gives the (model-implied) swap rate for any maturity. The maturity is measured in years and assumed to be continuous.

The following yield curve models are included:

  • Discrete model
  • Spline model
  • Vasicek model

The following numerical methods are used:

  • Newton solver
