Yield Curve Demo

Demonstrates methods to build a yield curve.

Demo

This interactive demo builds the US swap rate curve using three types of models: spline, Vasicek, and discrete. The fitted swap, zero-coupon, and forward curves are shown along with their bump response profile. Over 10 years of daily historical data are available for back-testing and calibration.

Docs

This section contains a mathematical description of the yield curve models and numerical methods used in this demo. In addition, for each model type we provide a summary of the features of different parameterizations. Links to wikipedia pages and references to published papers are (to be) included.

Code

The source code of the interactive demo as well as the web content is hosted on github under the MIT license. The demo is written entirely in Javascript and runs in the browser. A collection of tools and libraries are used, including bootstrap, canvasjs, jQuery, mathjax, numericjs, and typescript.
